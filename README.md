# miRNA-seq_benchmark
This repository contains R code for comparing normalization and differential expression (DE) methods  for miRNA sequencing data (miRNA-seq).


## Repository structure

```text

.
├── DE_benchmark/
│   ├── 1_generate_DE_results.R                 # Script that precomputes pairwise comparison results for all methods
│   ├── 2_benchmark_DE.R                        # Differential expression benchmarking script
│   ├── input_data/
│   │   ├── mature_sense_minExpr0_RCadj.mat                 # sRNAbench Read Count matrix for the benchmark study
│   │   ├── mature_sense_minExpr0_RCadj_libraryRPM.mat      # sRNAbench Read Per Million (normalized to miRNA mapping reads)
│   │   └── monotonic.csv
│   ├── pairwise_DE/        # Pairwise DE comparisons are stored here (generated by 1_generate_DE_results.R)
│   └── source/
│       └── utils.R         # Helper functions used by the main script
└── normalization/
    ├── 1_normalization_benchmark.R         # Normalization benchmarking script
    ├── R/
    │   └── normalization.R                 # Helper functions used by the main script
    ├── data/
    │   ├── mature_sense_minExpr0_RCadj.mat                 # sRNAbench Read Count matrix for the benchmark study
    │   └── mature_sense_minExpr0_RCadj_totalRPM.mat        # sRNAbench Read Per Million (normalized to all reads) 
    └── plots/                         # Generated by 1_normalization_benchmark.R

```

## Run the normalization benchmark

The `1_normalization_benchmark.R` script expects the helper module at `R/normalization.R` and reads its input files from `data/`. 
Navigate to `normalization` and run the script there so that these relative paths resolve correctly.
```bash
cd normalization
Rscript 1_normalization_benchmark.R
```
The script creates the normalization/plots/ directory and writes the generated figures in there.


## Input data

The analysis currently expects the following sRNAbench matrices:

```text
data/mature_sense_minExpr0_RCadj.mat
data/mature_sense_minExpr0_RCadj_totalRPM.mat
```

## Output

Plots for each benchmark are written to the `plots/` subdirectory, which is created automatically when the analysis runs.

## Configuration

Several options can be changed near the beginning of the main scripts:

```r
minRPM <- 50                # minimum average RPM required for a miRNA to be included in the analysis
resample_reads <- FALSE     # whether reads are resampled for the "Read Count" method
microRNAs_to_drop <- c()    # microRNAs to be removed prior to normalizations/transformations
mice_to_drop <- c()         # mice to be dropped from the analysis
```

These control the minimum expression threshold, optional read resampling, and the exclusion of selected miRNAs or mice.
